Error Control

Error detection and correction, often known as error control, is a technique used in information theory and coding theory with applications in computer science and telecommunication. It allows for reliable delivery of digital data through unstable communication networks. Channel noise affects many communication channels, resulting in errors being added during transmission from the source to the receiver. In many circumstances, error detection techniques allow for the detection of such errors, whereas error repair techniques allow for the restoration of the original data.
